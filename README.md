# 🕊️ X-Particle Forex Dataset — 1971

> *Dedicated to the martyrs of the Bangladesh Liberation War of 1971.*

This dataset contains comprehensive **Forex MT5 historical price data** with a
start date of **16 December 1971, 12:00 GMT+6** — Bangladesh's **Victory Day** —
chosen as a tribute to the freedom fighters and martyrs whose sacrifice gave birth
to an independent nation.

The name **X-Particle** symbolises the unknown variable in every equation — just
As the bravery of millions of unnamed martyrs remains the invisible force behind
Bangladesh's existence.

---

## 📦 Dataset Overview

| Field            | Details                                       |
|------------------|-----------------------------------------------|
| **Source**       | MetaTrader 5 (MT5) Historical Data            |
| **Start Date**   | 16 December 1971, 12:00 GMT+6 (Victory Day)  |
| **Asset Class**  | Foreign Exchange (Forex)                      |
| **Format**       | CSV                                           |
| **License**      | CC BY 4.0                                     |

- Backtesting and evaluating algorithmic trading strategies
- Training time-series forecasting models (LSTM, Transformer, N-BEATS, etc.)
